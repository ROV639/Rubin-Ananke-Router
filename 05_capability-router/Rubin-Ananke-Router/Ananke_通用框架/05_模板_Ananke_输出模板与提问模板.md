# 05 模板｜Ananke 输出模板与提问模板

本文件给你**可直接复制**的空白模板。每个模板都跟 02 的五层引擎和 04 的五面镜头一一对齐，填进去就能跑。

---

## 通用诊断模板（默认必用）

对应 02 的五层下钻 + 04 的五面镜头（代价 + 盲区必照，反证/执行/边界按场景追加）。

```markdown
结论：
[一句话判断，含置信度刻度：确定 / 基本确定 / 存疑 / 无法判断]

【L1 对象】
[你要审视的是什么]

【L2 真问题】
[用户说的 vs 实际上的]

【L3 原因】
[结构性原因；如适用，标注七宗罪中的某一面]

代价：
- [成本 1，带量级]
- [成本 2，带量级]
- [隐性成本]

盲区：
- [用户没看到的]
- [你本次判断中可能也漏掉的]

反方：
[什么证据会推翻这个结论；找不到反证时，说明找了哪些源、什么范围没找到]

边界（如适用）：
- [ ] 涉及发布 / 删除 / 付款 / 远程？
- [ ] 不可撤回？止损口在哪？

置信度：[确定 / 基本确定 / 存疑 / 无法判断]
依据：[T1/T2/T3/T4/T5，具体哪几层，来源日期]

下一步：
1. [具体动作 + 验证方式；今天/本周能动手]
2. [可选第二动作]
```

## 信息不足专用模板（不降级，专用格式）

```markdown
现在不能判。

缺的关键事实：
- [ ] [缺失项 1]
- [ ] [缺失项 2]

我只需要你补：
[一个问题，只问会改变判断的那个；不要为了礼貌多问]
```

---

## 投资/市场类强制字段（在通用模板基础上追加）

```markdown
事实来源：
- [来源 1 + 日期]
- [来源 2 + 日期]

失效条件：
[什么新信息会让这个判断作废]

风险：
[具体风险，不写"市场有风险"这种废话]

不构成投资建议：是
```

---

## 执行困境类强制字段（在通用模板基础上追加）

```markdown
当前最大卡点：
[一句话点出真正的阻碍]

不是问题的问题：
[用户在担心但其实可以绕过的东西]

必须放弃的东西：
[继续推进就要放掉的某个选项/执念]

今天能做的一步：
[今天能做的最小动作]
```

---

## 反证检查专用模板（当你只想要"挑战现有结论"时单独跑）

```markdown
当前结论：
[一句话复述要挑战的判断]

必须推翻它的证据：
1. [来源 A + 数据 + 日期]
2. [来源 B + 反例 + 日期]

我找过但没找到反方的范围：
[搜索范围、时间窗、来源类型]

更新后的结论：
[结论调整 / 不变 / 反转]

置信度：[调整后刻度]
```

## 空白信息不足提问模板（极简）

```markdown
现在不能判。

缺：[一句话]

我只需要你补：[一个问题]
```

---

## 使用规则

1. 默认从"通用诊断模板"起手。只有信息真的不够才换"信息不足专用模板"。
2. 投资/市场和执行困境在通用模板后**强制追加**对应字段，不能省。
3. 只想要"挑战现有结论"时用反证检查模板，单独跑一次比硬塞在通用模板里更清晰。
4. 模板是骨架，不是约束。出现模板里没有的字段（如"必须放弃的东西"在投资判断里不适用），就跳过；不要硬填。
