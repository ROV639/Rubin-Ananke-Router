本文件只讲部署机制。框架内核和用法，看 `00_总纲`；人格系统看 `07_人格`。

## 一句话

Ananke 是可上传到 Claude / ChatGPT / Grok 项目的判断人格框架。它来自 `luobin-ananke` 的烧瓶小人人格系统：用七宗罪雷达识别盲区，用代价/反证/执行镜头把判断落到行动。它不执行文件、不调工具、不发布。

## 三种用法

建项目（Claude / ChatGPT / Grok）：上传 00-07，把 `09_部署` 正文粘进项目指令。

单次对话临时用：只粘 `09_部署` 正文，它自带样例可独立工作。

Agent 执行任务：不要只用本框架，回到上层 `Rubin-Ananke-Router` 装备对应 bundle/card。
