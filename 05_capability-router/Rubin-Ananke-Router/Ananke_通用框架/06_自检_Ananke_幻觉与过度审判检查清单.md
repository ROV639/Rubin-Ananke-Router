## 模板结构自检

- [ ] L1（明确对象）有没有先写？没有就补——说不清对象就别判。
- [ ] L2（真问题）有没有翻到表层下面一层？
- [ ] L3（结构性原因）有没有区分一次性失误 / 系统性偏差 / 不可抗力？
- [ ] L4（代价）有没有带量级？（不允许"可能有代价"这种空话）
- [ ] L5（下一步）是不是 1-3 个具体动作？每个有没有"今天/本周能动手 + 验证方式"？

## 镜头覆盖自检

- [ ] 代价镜头和盲区镜头照了没？（默认必照，缺一即不合格）
- [ ] 场景需要的追加镜头照了没？
  - 市场/策略/投资判断 → 反证镜头
  - 拖延/分散/执行困境 → 执行镜头
  - 涉及发布/删除/付款/远程 → 边界镜头
- [ ] 反证镜头有没有明确"找了哪些范围没找到"，而不是"目前没有明显反方"就完事？

## 口吻自检（防止过度审判）

- [ ] 没把情绪词放大判断（"显然""绝对""完全"这种词少用）。
- [ ] 没把人格攻击包装成诊断。
